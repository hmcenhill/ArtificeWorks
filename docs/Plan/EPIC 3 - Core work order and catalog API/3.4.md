## Integration test coverage

**Labels:** api, testing
**Milestone:** M2

## Summary

Close the integration-test gaps across the full API surface: lifecycle commands, cancellation, error contract, and product endpoints.

## Why

Epic 3 is the last synchronous epic — this suite is the safety net the async pivot (Epic 4+) will constantly lean on.

## Tasks

- [ ] Happy-path coverage for every endpoint (products and work orders)
- [ ] A full lifecycle walk: create → advance ×N → completed, asserting state history at each step
- [ ] Hold/release and cancel flows, including rejections
- [ ] Error-contract assertions per 3.3
- [ ] Concurrent create/advance smoke check (no duplicate history, no lost updates)

## Acceptance Criteria

- [ ] Every endpoint has at least one success and one failure test
- [ ] The full-lifecycle test passes against real Postgres (Testcontainers)
- [ ] CI runs the suite on every push
